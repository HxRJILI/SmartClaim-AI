# Chat Assistant

SmartClaim AI includes a context-aware AI chat assistant.

## Features

### Role-Based Responses
The assistant adapts responses based on user role:
- Workers: Guidance and status
- Managers: Insights and recommendations
- Admins: System information

### RAG Integration
- Retrieves relevant policy documents
- References historical tickets
- Provides accurate information

### Multi-Language
- French and English support
- Automatic language detection

## Usage

### Starting a Conversation
1. Click Chat icon
2. Type your question
3. Receive AI response

### Example Questions

**Workers:**
- "How do I report a safety incident?"
- "What's the status of my ticket?"
- "What's the policy for X?"

**Managers:**
- "Show department performance"
- "What tickets need assignment?"
- "Generate weekly report"

**Admins:**
- "System health status?"
- "Show cross-department trends"
- "Which AI service is slowest?"

## Conversation History

- History maintained per session
- Context preserved across messages
- Clear history option available
