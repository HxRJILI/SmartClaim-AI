# Ticket Submission

Guide to submitting tickets in SmartClaim AI.

## Input Methods

### Text Input
1. Navigate to New Ticket
2. Select Text tab
3. Enter description in French or English
4. Submit for AI classification

### Image Upload
1. Select Image tab
2. Drag and drop or click to browse
3. Preview the image
4. Submit for visual analysis

### Voice Recording
1. Select Audio tab
2. Click microphone to start
3. Speak your description
4. Click stop when finished
5. Review transcription
6. Submit

### Document Upload
1. Select Document tab
2. Upload PDF, Word, or Excel
3. Review extracted text
4. Submit

## AI Classification

After submission, the AI will:
1. Analyze your input
2. Classify into category
3. Assign priority
4. Generate summary
5. Predict SLA

## Review and Submit

1. Review AI classification
2. Modify if needed
3. Add additional notes
4. Submit ticket
5. Receive confirmation
