# Dashboards

SmartClaim AI provides role-specific dashboards for different user types.

## Worker Dashboard

The worker dashboard provides a simplified interface for ticket submission and tracking.

### Components

#### My Tickets
- List of submitted tickets
- Status indicators
- Quick filters

#### New Ticket
- Multimodal input form
- AI classification preview
- Submit button

#### Chat Assistant
- AI chat interface
- Quick questions
- Ticket status inquiries

### Navigation
- Home / Dashboard
- My Tickets
- New Ticket
- Chat
- Profile

## Manager Dashboard

The manager dashboard provides department oversight and team management.

### Components

#### Department Queue
- All department tickets
- Priority sorting
- Assignment controls

#### Team Performance
- Resolution metrics
- SLA compliance
- Workload distribution

#### Analytics
- Trend charts
- Category breakdown
- Time-to-resolution

### Actions
- Assign ticket
- Change priority
- Update status
- Generate report

## Admin Dashboard

The admin dashboard provides full system access and configuration.

### Components

#### System Overview
- Cross-department metrics
- AI service health
- User statistics

#### User Management
- User list
- Role assignment
- Department assignment

#### Settings
- System configuration
- Feature flags
- Integration settings

### Monitoring
- AI service status
- Database health
- Error logs
