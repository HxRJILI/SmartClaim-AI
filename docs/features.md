# Features

SmartClaim AI includes a comprehensive set of features for industrial incident management.

## Multimodal Input Processing

### Text Input
- Free-form text descriptions
- French and English language support
- Automatic spell-check suggestions

### Image Upload
- Drag-and-drop interface
- Supported formats: JPG, PNG, WebP, HEIC
- Maximum file size: 10MB
- Automatic image optimization

### Voice Recording
- In-browser recording
- Supported formats: MP3, WAV, M4A, OGG
- Maximum duration: 5 minutes
- Two-stage ASR (VOSK + Whisper)

### Document Upload
- PDF with OCR support
- Microsoft Office (DOCX, XLSX, PPTX)
- Plain text and Markdown
- CSV and structured data

## AI-Powered Classification

### Automatic Categorization
- Safety incidents
- Quality defects
- Maintenance issues
- Logistics problems
- HR matters

### Priority Assignment
- Critical: < 1 hour response
- High: < 4 hours response
- Medium: < 24 hours response
- Low: < 72 hours response

### Confidence Scoring
- 90-100%: Auto-processed
- 70-89%: Flagged for review
- Below 70%: Human validation required

## Visual Evidence Analysis

The Vision Language Model (VLM) analyzes images to extract:
- Scene type identification
- Equipment recognition
- Hazard detection
- Safety compliance checks
- Severity assessment

## Predictive SLA Engine

AI-powered resolution time prediction:
- Historical pattern analysis
- Category-based estimation
- Department workload consideration
- Breach probability calculation

## AI Chat Assistant

Context-aware conversational support:
- Role-specific responses
- Policy information retrieval
- Ticket status inquiries
- Guided ticket submission

## Analytics & Reporting

### Worker Analytics
- Personal ticket history
- Resolution times
- Status tracking

### Manager Analytics
- Department performance
- Team workload
- SLA compliance
- Trend analysis

### Admin Analytics
- System-wide metrics
- Cross-department comparison
- AI service health
- Usage statistics
