# Changelog

All notable changes to SmartClaim AI.

## [1.0.0] - 2025-01-15

### Added
- Initial release of SmartClaim AI
- Multimodal ticket submission (text, image, audio, document)
- AI-powered classification using Google Gemini
- Vision Language Model integration for image analysis
- Two-stage audio transcription (VOSK + Whisper)
- Predictive SLA engine
- Role-based access control (Worker, Manager, Admin)
- AI chat assistant with RAG integration
- Department management
- User management
- Analytics dashboards
- Professional documentation

### Technical
- Next.js 15.x with App Router
- Supabase for database and authentication
- 8 AI microservices architecture
- Docker Compose deployment
- Read the Docs documentation
