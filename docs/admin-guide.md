# Administration Guide

Guide for SmartClaim AI administrators.

## User Management

### Creating Users
1. Navigate to Admin > Users
2. Click "Add User"
3. Enter email and details
4. Assign role and department
5. Save

### Role Assignment
- Worker: Basic access
- Manager: Department oversight
- Admin: Full access

### Department Assignment
1. Select user
2. Choose department
3. Save changes

## Department Management

### Creating Departments
1. Navigate to Admin > Departments
2. Click "Add Department"
3. Enter name and description
4. Assign manager
5. Save

### Managing Departments
- Edit department details
- Change manager
- View department statistics
